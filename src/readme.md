#### This folder contains the `main.py` analysis script that performs **both** analyses:


1. **Global Analysis (1950–2023)**  
   - World and continental trends  
   - Top 15 countries by life expectancy gain  
   - Catch-up effect and convergence visualization  
   - Sub-Saharan Africa vs rest of the world (HIV/AIDS impact)  
   - Major health shocks (Great Chinese Famine 1959–1961, COVID-19)

2. **Italy Deep Dive**  
   - Italy’s trajectory compared to:  
     • Western Europe  
     • G7  
     • Eastern Europe  
     • Mediterranean countries  
     • Americas  
     • BRICS+  
     • World  
   - Detailed tables and rankings

Features:
- **All charts** in `../results/img/` with descriptive filenames
- Prints key findings and summary tables in the console
